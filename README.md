## About Me

I'm a backend developer focused on building production-quality systems using the Java/Spring Boot ecosystem. My work primarily spans fintech infrastructure, including high-concurrency trading platforms and intelligent financial analysis engines. I emphasize secure API design, atomic transaction handling, and clean architecture (SOLID) to ensure long-term system maintainability.

- Building high-concurrency trading backends and automated payment handlers
- Expert in Spring Security, JWT authentication, and Role-Based Access Control (RBAC)
- Focused on data integrity, JPA/Hibernate optimization, and secure financial workflows
- Based in India
